# Roadmap

This roadmap reflects current direction and may change based on gameplay feedback.

## Next Up

- Advanced tracker layout presets (compact, split, raid-lead views)
- Additional affix context (pull-impact hints/tags)
- Optional combat-only minimal mode
- Better objective row compaction on small frame sizes

## Planned Quality Improvements

- More resilient API fallback handling across patches
- Better per-dungeon data validation
- Expanded profile migration handling for future schema changes
- Additional UI polish for high-scale and ultra-wide setups

## Community-Driven Features

- Shared theme packs
- One-click profile import from short links
- More font/texture style packs

## Long-Term

- Optional modular plugin-style display blocks
- Multi-profile auto-switch by spec/role
- Localization support
